### 🙍‍♂️ React User Register

## 🚀 How to execute this project:

Make sure to have all dependencies installed:
```
npm i
```
Enter in the Backend directory and execute: 
```
npm start
```
After enter in the Frontend directory and execute: 
```
npm start
```
